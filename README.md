# Sentiment-Analysis-Navie-Bayes

# Overview
This project performs sentiment analysis on Twitter data and classifies tweets into Positive, Negative, and Neutral categories using NLP techniques.

# Features

- Text preprocessing and cleaning
- TF-IDF / CountVectorizer
- 3-class sentiment classification
- Rule-based neutral label generation using VADER

# Tech Stack

- Python
- NLP (Scikit-learn, NLTK)
- VADER Sentiment Analyzer
